## 已交付范围

ASR Provider：

- Groq Whisper
- Qwen Local 0.6B：固定模型 `Qwen/Qwen3-ASR-0.6B`，默认 `http://127.0.0.1:8000/v1`，仅允许回环地址，无 API Key，不自动降级到云端。
- Qwen API 1.7B：固定模型 `Qwen/Qwen3-ASR-1.7B`，仅允许 HTTPS，Bearer Key 可选。
- Doubao：新版单 `X-Api-Key`、Flash 录音文件接口、固定资源 `volc.bigasr.auc_turbo` 和模型 `bigmodel`。
- OpenAI Realtime：实际连接模型记录为 `gpt-realtime-2`，转录子模型为 `gpt-4o-mini-transcribe`。

Rewrite Provider：

- OpenAI
- DeepSeek V4 Flash/Pro：Flash 关闭 Thinking，Pro 开启 Thinking；批量和 SSE 只输出 `content`，忽略 `reasoning_content`。

运行规则：

- Provider 选择是唯一路由真相，任何失败都不得跨 Provider 自动回退。
- 旧 `streamingMode=true` 首次读取迁移为 `openai-realtime`，旧 Streaming 开关已移除。
- OpenAI Realtime 遇到 Speak to Edit 选中文本时明确拒绝，不回退 Groq。
- API Key 存在 Windows Credential Manager；配置导出不包含 Key。
- Privacy 页面只保留真实生效的 `Save raw audio`。
- Qwen Local 服务由用户在独立 Python 3.12 环境中安装、启动和更新；应用不下载模型、不管理本地服务。
